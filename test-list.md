## 📋 체크리스트 사용법

1. 테스트 완료 시: `- [ ]` → `- [x]`
2. 실패 시: 뒤에 `❌ 실패: 에러 메시지` 추가
3. 스킵 시: `⏭️ 스킵: 이유` 추가

**예시**:
- [x] **jpg → png** (우선순위: 10 ⭐) `✅ 완료 (3.2초)`
- [ ] **heic → jpg** (우선순위: 9 ⭐) `⏭️ 스킵: heic 파일 없음`
- [ ] **gif → mp4** (우선순위: 8 ⭐) `❌ 실패: FFmpeg 로딩 에러`
